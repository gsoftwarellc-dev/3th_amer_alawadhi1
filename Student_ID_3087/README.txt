HOW TO RUN
------------------------------------------------------------------------
1) Open a terminal inside the Website/ folder.

2) Run:
     chmod +x start.sh
     ./start.sh

   This installs anything missing (Docker, .NET SDK), sets up SQL Server
   and the database, and launches the website. First run takes a few
   minutes; later runs are fast.

3) When you see "Now listening on: http://localhost:5188", open that
   URL in Google Chrome.

4) Press Ctrl+C in the terminal to stop the website.
